# Works Data Comments

Originally derived from the `works.jsonc` comments.

- `profile`: プロフィール情報
- `organizations`: 所属組織リスト
- `works`: 制作物リスト

## Works Fields (作品情報のフィールド)
- `title`       : 作品名
- `description` : 説明文
- `tags`        : タグ配列 (フィルタリングに使用)
- `url`         : デモURL (省略可)
- `github`      : GitHubリポジトリURL (省略可)
- `detailUrl`   : 詳細ページURL (省略可、指定すると「詳しく見る」ボタンが表示される)
- `thumbnail`   : サムネイル画像URL (省略可、SHOW_THUMBNAILSフラグで制御)
- `date`        : 制作年月
